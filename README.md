#### 软件架构
软件架构说明
1、核心语言: Java
2、GUI库: JavaFX
3、图形和动画: JavaFX内置
4、网络通信: Java Socket API (联机模式)
5、数据存储: SQLite (记录分数和玩家进度)

#### 安装教程

1、确保你的系统已安装Java JDK 8及以上版本。
2、下载项目源代码或可执行文件。
3、如果下载的是源代码，使用JavaFX SDK来编译和运行。
4、如果下载的是可执行文件，则直接双击运行。

#### 使用说明

1、启动游戏后，选择单机模式或联机模式。
2、对于联机模式，请确保网络连接正常。
3、使用鼠标进行卡牌操作。
4、游戏目标和规则根据具体的纸牌游戏而变。
